# Phase 2: 敵4種＋弾＋ガード＋危険物

## 前提

- 依存Phase:
  - Phase 1
- 参照:
  - `docs/00_PROJECT_SPEC.md`

## このPhaseのゴール

深層坑道に潜む危険生物・旧坑道機械・追跡型脅威を実装し、採掘中に戦闘と危険回避が発生する状態を作る。
敵は単なる障害物ではなく、潜行失敗・救助リスク・帰還判断に直結する存在として機能させる。

また、ソナーが敵反応も拾うことで、「何かいるが敵か回収ポイントか分からない」緊張感を実現する。

## 完了条件チェックリスト

### 敵
- [ ] 敵4種が実装されている
- [ ] 深度に応じて出現テーブルが変わる
- [ ] 敵サイズが3×3〜5×5セルで管理される
- [ ] プレイヤーの進行を妨げる
- [ ] 倒す / 避ける判断が発生する

### 弾
- [ ] 射撃型・拡散型などの弾が実装される
- [ ] 弾はグリッドまたは疑似グリッド上で移動する
- [ ] プレイヤー被弾判定が機能する
- [ ] ガードで防げる弾がある

### ガード
- [ ] 長押し / Space でガードできる
- [ ] ガード中は移動停止
- [ ] ガードで一部弾・衝突ダメージを軽減できる
- [ ] ガードしすぎると時間損失になり、潜行効率に影響する

### 危険設計
- [ ] 敵が潜行失敗や帰還困難に直結する
- [ ] 帰還中に脅威が増す展開がある
- [ ] ソナーで敵反応が拾える
- [ ] 敵と回収ポイントやアイテムの反応が曖昧に混ざる場面がある

### 報酬
- [ ] 敵を倒すとゴールドやアイテムが落ちる
- [ ] 敵処理がリスクに見合うリターンを持つ

## 敵4種

### 1. 坑道生物
- サイズ: 3×3
- 特徴: 近接で突っ込んでくる
- 役割: 基本近接敵

### 2. 射撃型危険物
- サイズ: 4×4
- 特徴: 一定間隔で直線弾
- 役割: 遠距離圧力

### 3. 拡散型危険物
- サイズ: 5×5
- 特徴: 8方向拡散弾
- 役割: 通路制圧

### 4. 追跡型
- サイズ: 3×3
- 特徴: 帰還時にしつこく追う
- 役割: 欲張りプレイへの圧力

## Bullet仕様

- 基本速度: 1セル/0.05秒
- 有効距離: 16セル程度
- ブロック接触で消滅
- ガード判定あり
- 8方向発射可
