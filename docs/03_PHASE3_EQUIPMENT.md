# Phase 3: 装備＋レアリティ＋掘削幅変化＋装備画面

## 前提

- 依存Phase:
  - Phase 2
- 参照:
  - `docs/00_PROJECT_SPEC.md`

## このPhaseのゴール

採掘物や敵ドロップから装備を入手し、プレイヤーが掘削効率・生還率・探索性能を強化できるようにする。
装備は借金返済効率に直結し、単純な火力だけでなく、ソナー・酸素効率・救助コスト軽減などにも影響する。

## 完了条件チェックリスト

### 装備
- [ ] 掘削具 / 防具 / アクセサリ / 靴の4カテゴリがある
- [ ] レアリティがある
- [ ] 深度に応じて性能が変化する
- [ ] 装備で掘削幅が変化する
- [ ] 装備で移動・掘削速度が変わる
- [ ] 装備で防御・酸素・ソナー補助などが変化する

### UI
- [ ] 装備画面がある
- [ ] 所持装備と装備中アイテムが確認できる
- [ ] 比較表示ができる
- [ ] レアリティが分かる

### ドロップ
- [ ] 敵や鉱石から装備ドロップする
- [ ] 深いほど良い装備が出やすい
- [ ] レアリティに応じた見た目や色がある

## 装備カテゴリ

### 掘削具
- 攻撃力
- 掘削速度補正
- 掘削幅
- 掘削形状

### 防具
- 被ダメ軽減
- HP増加
- 状態悪化軽減

### アクセサリ
- ソナー補助
- ゴールド取得補正
- ドロップ率補正
- 回収費軽減
- 借金返済補正

### 靴
- 移動速度
- 方向転換補助
- 帰還性能補助

## レアリティ

| レアリティ | ドロップ率 | 基礎倍率 | 効果数 |
|---|---|---|---|
| Common | 高 | ×1 | 0〜1 |
| Uncommon | 中 | ×3 | 0〜2 |
| Rare | 低 | ×10 | 1〜2 |
| Epic | かなり低い | ×50 | 2〜3 |
| Legendary | 非常に低い | ×500 | 2〜3 |
| Cursed | 極低 | ×5000 | 3 + デメリット |

## 掘削具の追加パラメータ

| パラメータ | 説明 | 範囲 |
|---|---|---|
| 攻撃力 | 敵や硬い壁への有効性 | 10〜10000 |
| 掘削速度補正 | 移動/掘削速度補正 | 0.5〜3.0 |
| 掘削幅 | 同時に掘る幅 | 5〜11 |
| 掘削形状 | Square / Diamond / Fan | 3種 |

## 特殊効果例

- ソナー探知距離+X
- ソナー方向精度+X
- 回収ポイント探知強化
- 酸素効率+X%
- フィルター劣化軽減
- 救助費-X%
- ドロップ率+X%
- ゴールド取得+X%
- 深層ノイズ耐性+X
