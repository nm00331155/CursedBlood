# Phase 7: 実績＋恒久研究報酬＋ランキング

## このPhaseのゴール

潜行記録・借金返済・深度到達・生還率などに基づく実績を実装し、恒久強化や称号要素を追加する。

### 掘削系
- 初めて100m到達

- 1回で大量掘削
- 硬岩大量破壊

### 戦闘系
- 敵累計撃破
- ノーダメ潜行
- ボス撃破

### 経済系
- 借金を一定額返済
- 単回利益達成

### 生還系

- 回収ポイント帰還回数
- 地上自力帰還回数

- ソナーで回収ポイント発見
- 深層調査達成
- 特殊資源回収
