# Phase 6: 深層大型脅威＋アリーナ＋高報酬エリア

## 前提

- 依存Phase:
  - Phase 2
- 参照:
  - `docs/00_PROJECT_SPEC.md`

## このPhaseのゴール

深層に到達したプレイヤーに対して、大型脅威との戦闘を実装する。
旧仕様の「魔王」ではなく、深層大型生物・封鎖区画の主・高価値資源守護体などとして再構成する。
ボス戦は危険だが、その分高い報酬や深層進行の節目になる。

## 完了条件チェックリスト

- [ ] 深度条件で大型脅威が出現する
- [ ] ボスサイズが20×20セル相当で表現される
- [ ] 周囲がアリーナ化される
- [ ] HPを持つ
- [ ] 複数攻撃フェーズがある
- [ ] 撃破報酬が高い
- [ ] 再戦可能または深層進行条件になっている

## ボスの位置づけ

候補:
- 深層大型生物
- 封鎖坑道の主
- 高価値鉱脈を守る変異体
- 旧坑道の大型警備機構

## ボス仕様

- サイズ: 20×20セル
- 周囲40×40セル程度をアリーナ空洞化
- HPバー表示
- 3フェーズ攻撃

### フェーズ例
- Phase1: 直線弾 / 十字弾
- Phase2: 8方向弾 / 拡散弾
- Phase3: 全方位弾 + 落盤警告

## 報酬
- 高額ゴールド
- Rare以上確定ドロップ
- 深層進行フラグ
- 調査ポイントや契約解放
