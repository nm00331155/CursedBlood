# Phase 9: スキル＋演出＋バランス調整＋タイトル画面

- 依存Phase:
  - Phase 8
- 参照:

## このPhaseのゴール

ここまでの全システムを統合し、演出・スキル・入力感・バランス・UI・タイトル画面を磨き上げる。
特に本作では「掘削の気持ちよさ」「カツカツ感」「帰還判断の緊張感」「借金返済の圧力」が重要なため、その体験を強化する。

## 完了条件チェックリスト

- [ ] スキルが実装される
- [ ] ソナー演出が強化される
- [ ] 酸素 / フィルター劣化演出が入る
- [ ] 回収成功 / 救助失敗の演出が強化される
- [ ] 入力感が改善される
- [ ] バーチャルパッドが磨かれる
- [ ] タイトル画面が完成する
- [ ] バランスが一通り調整される
