## コンセプト

呪われた血脈が世代を重ねて地下深くの魔王を目指す掘削ハクスラ。
一族に「35歳で死ぬ呪い」がかけられている。初代は魔王との契約で莫大な借金も背負っている。
1プレイ＝1人生＝60秒。死んだら次世代に遺品・遺産を託して再スタート。
世代を重ねるほど装備もステータスもインフレし、到達深度が伸びる。

キーワード: ハクスラ、ローグライク、ハイパーインフレ、周回、スコアアタック、短命、呪い、借金

## グリッド設計（小ブロック方式）

- **セルサイズ**: 16×16px
- **列数**: 67列（1080px / 16px = 67.5 → 67列、左右マージン各4px）
- **画面表示行数**: 87行（1400px / 16px = 87.5 → 87行）
- **行数**: 無限（下方向に動的生成）
- キャラ移動でカメラ追従、新しい行を下から動的生成、上の行は破棄
- **チャンク管理**: 16行を1チャンクとして管理。画面外のチャンクは破棄しメモリを節約

### 深度の単位変換

旧仕様の「深度1」（1ブロック=140px）は新仕様では約8.75セル（140/16）に相当する。ゲーム内の「深度」表示はメートル単位とし、ピクセル位置から算出する。

```
深度(m) = プレイヤーY座標(px) / 16
```

旧仕様の「深度100」（ボス出現）は新仕様では「深度875m」に相当する。切りの良い数字にするため、ボスは深度1000mごとに出現とする。

## 移動と掘削

### 8方向移動

| 方向 | ベクトル | キーボード |
|------|------|------|
| 上 | (0, -1) | ↑ |
| 下 | (0, 1) | ↓ |
| 左 | (-1, 0) | ← |
| 右 | (1, 0) | → |
| 左上 | (-1, -1) | ← + ↑ |
| 右上 | (1, -1) | → + ↑ |
| 左下 | (-1, 1) | ← + ↓ |
| 右下 | (1, 1) | → + ↓ |

- セル単位移動。基本速度 0.02秒/セル（50セル/秒）
- 装備で最短 0.005秒/セル（200セル/秒）まで短縮可
- 移動速度 = 基本速度 × 掘削具補正 × ブロック硬度
- 方向転換は即時。現在セル移動完了後に新方向へ進む（先行入力バッファあり）
- 斜め移動時の速度補正はなし（8方向全て同速度。斜めが有利になるのは意図的なゲーム性）

### 掘削幅

掘削具の「幅」パラメータにより、進行方向に対して垂直方向に同時に掘る範囲が変わる。

```
掘削幅 = 掘削具の幅パラメータ（1〜11、奇数のみ）
```

例: 幅5の場合、進行方向に対して左右2セルずつ（計5セル幅）を同時に破壊。

```
進行方向: →
  □□□□□
  □□□□□  ← 5セル幅で掘削
  □□★□□  ← ★=プレイヤー中心
  □□□□□
  □□□□□
```

斜め移動時の掘削幅も同様に、進行方向に垂直な方向に展開する。

### プレイヤーの物理サイズ

プレイヤーは5×5セル（80×80px）の占有サイズを持つ。当たり判定もこのサイズ。これにより、幅5セル以上のトンネルでなければ通過できない。掘削幅1の装備では「自分の体幅分は自動で掘れる（幅5の最低保証）」とする。

```
実効掘削幅 = max(掘削具の幅パラメータ, 5)
```

高レアリティの掘削具は幅7, 9, 11と広がり、広大なトンネルを一気に掘れる。

## 敵設計（4種 + ボス）

敵は複数セルを占有する。

| 種類 | サイズ | 攻撃 | 硬さ | 備考 |
|------|------|------|------|------|
| スライム型 | 3×3セル(48px) | 弾なし | 2 | 序盤用 |
| 射撃型 | 4×4セル(64px) | 2秒間隔で1方向に1発 | 3 | |
| 拡散型 | 5×5セル(80px) | 3秒間隔で8方向同時に弾 | 4 | |
| 自爆型 | 3×3セル(48px) | 隣接で3秒カウントダウン→周囲10セル大ダメージ | 2 | |

弾: セル上を直線移動（1セル/0.05秒 = 20セル/秒、16セルで消滅）。ブロックに当たると消滅。8方向に発射可能。

深度帯ごとに色替え（パレットスワップ）で4色展開。

## ボス（深度1000mごと）

- 20×20セル（320×320px）
- 周囲が自動空洞化してアリーナ形成（40×40セル = 640×640px）
- HP表示、プレイヤーがボスに隣接してスワイプ方向がボス方向なら攻撃判定
- 弾幕フェーズ:
  - Phase1（HP100〜60%）: 十字4方向に2発ずつ、3秒間隔
  - Phase2（HP60〜30%）: 8方向に1発ずつ、2秒間隔
  - Phase3（HP30〜0%）: 全方向弾 + ランダム列に崩落警告（1.5秒後ダメージ）
- 倒すとRare以上確定ドロップ。迂回も可能だが報酬なし
- **ボスドロップテーブル**: Rare 70%, Epic 20%, Legendary 8%, Cursed 2%

## 1プレイの時間構成（60秒＝35年）

| 秒数 | 年齢 | フェーズ | 特徴 |
|---|---|---|---|
| 0〜20秒 | 0〜12歳 | 少年期 | 体が小さい（占有3×3セル）、掘削遅い、敵弱い。「才能の種」系アイテムで成長率が決まる |
| 21〜45秒 | 13〜24歳 | 青年期 | 占有5×5セル、ステータスピーク、装備ドロップ豊富、稼ぎ時。敵も急に強くなる |
| 46〜60秒 | 25〜35歳 | 晩年期 | 占有5×5セル、「呪いの侵食」画面端から黒霧演出、ステータス徐々に低下、「遺言スキル」解禁 |

## 借金システム

- 初代借金: 100,000G、世代ごとに利息10%加算
- リザルト画面で「全額返済/半額返済/最低返済（利息分のみ）/返済しない」の4択
- 返済しないと借金取り敵の出現頻度UP
- 完済で永続「解放ボーナス: 全ステ×1.2」

## スコア計算

```
スコア = 最大深度(m) × 倒した敵数 × (1 + コンボ最大値/100) × (1 + 借金返済率) × 世代ボーナス
世代ボーナス = 1 + (世代数 × 0.05)
コンボ: 敵連続撃破で加算、3秒間撃破なしでリセット
```

## 記録システム

- **家系図**: 各世代の名前（ランダム生成・男女対応）、最大深度(m)、死因、使用掘削具、スコア、撃破数、返済額を記録
- **実績（20個）**: 解除で永続パッシブ付与。掘削系5個、戦闘系5個、収集系5個、世代系5個
- **ランキングボード（ローカル）**: 歴代最大深度TOP10、最高スコアTOP10、最速深度1000m到達

## パフォーマンス要件

小ブロック方式で60fps維持のための設計ルール:

- **チャンク管理**: 16行を1チャンクとし、画面外チャンクは即座に破棄する
- **描画最適化**: 空洞セルは描画しない。同じ種類の隣接セルはバッチ描画する
- **データ構造**: セルデータは byte 配列（1セル=1バイト）で管理。CellDataクラスのインスタンスは生成しない
- **当たり判定**: グリッド座標による O(1) ルックアップ。物理エンジンは使わない
- **更新範囲**: プレイヤー周囲のみ更新（掘削判定は移動先周辺のみ）
- **メモリ**: 67列 × 16行/チャンク × 1byte = 1,072 bytes/チャンク。画面内6チャンク + バッファ2チャンク = 約8.5KB
