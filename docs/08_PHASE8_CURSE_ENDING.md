# Phase 8: 深層調査＋完済後要素＋エンディング

## 前提

- 依存Phase:
  - Phase 7
- 参照:
  - `docs/00_PROJECT_SPEC.md`

## このPhaseのゴール

旧仕様の「呪い研究」「呪いエンディング」を廃止し、
借金完済後に開く本格的な深層調査・高難度契約・最深層の真相解明を実装する。
借金完済をゲームの終わりではなく中間地点とし、その先の最終目標とエンディングを成立させる。

## 完了条件チェックリスト

- [ ] 借金完済後に新コンテンツが解放される
- [ ] 深層調査または高難度契約が実装される
- [ ] 最終目標が提示される
- [ ] エンディングがある
- [ ] 真エンド条件がある

## 完済後の流れ

### 借金完済後に解放
- 深層ライセンス
- 特別契約
- 深層封鎖区画
- 最深層調査
- 高額維持費を伴う新装備領域

### 新目標例
- 両親の借金の真相を知る
- 最深層の超高価値資源を回収する
- 封鎖区画の主を倒す
- 巨大契約を完遂する

## エンディング案

### 通常エンド
- 借金完済
- 生活再建開始
- しかし深層への道はまだ残る

### 深層到達エンド
- 最深層調査成功
- 借金の真相・地下の秘密を知る

### 真エンド
- 完済 + 最深層制覇 + 特殊条件達成
- 真相解明と大団円 / あるいは次の挑戦への余韻
