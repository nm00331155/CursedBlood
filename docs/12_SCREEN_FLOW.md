# 画面遷移フロー全体図

## 前提

本仕様は「借金返済を目的に危険な坑道へ短時間潜行する」新設定に基づく。
旧仕様の呪い・世代交代・家系図前提の画面構成は採用しない。

## 主要画面一覧

1. タイトル画面
2. 主人公選択画面
3. 拠点 / 準備画面
4. 装備画面
5. 研究画面
6. 潜行開始
7. メインゲーム画面
8. 潜行結果画面
9. 借金返済画面
10. 潜行記録画面
11. 実績画面
12. ランキング画面
13. 設定画面
14. 完済後解放画面
15. エンディング / 真エンド画面
